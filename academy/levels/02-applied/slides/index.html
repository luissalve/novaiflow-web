Seis pasos para una plantilla sólidaSix steps to a solid template

  1. Elijan la tarea repetida.Pick the repeated task. Un correo, un resumen, una tabla que hacen seguido.An email, a summary, a table you do often.
  2. Escriban el prompt base.Write the base prompt. Rol · Tarea · Contexto · Formato.Role · Task · Context · Format.
  3. Marquen las variables.Mark the variables. Lo que cambia cada vez → {así}.What changes each time → {like this}.
  4. Prueben con 2 casos reales.Test with 2 real cases. ¿Sirve sin retoques grandes?Does it work without big edits?
  5. Agreguen una nota de uso.Add a usage note. Cuándo usarla y qué revisar antes de enviar.When to use it and what to check before sending.
  6. Guárdenla en el kit.Save it to the kit. Con nombre claro para que todos la encuentren.With a clear name so everyone finds it.
