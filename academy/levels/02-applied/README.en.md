## Learning objectives

By the end of this level, each participant will be able to:

1. **Explain** the difference between a one-off prompt and a reusable template.
2. **Convert** a working prompt into a template using `{variables}`.
3. **Recognize and use** the core prompt patterns: instruction, example, step-by-step, role, refine.
4. **Apply** four high-value template types: email, summary, spreadsheet, image.
5. **Choose** sensibly when a "most capable" model or mode is worth using.
6. **Test** a template against real cases and write a short usage note.
7. **Assemble** a shared **team template kit** with their colleagues.
8. **Practice** responsible use: human review, no sensitive data, verify facts.

## Responsible AI in this level

Every level reinforces the same principles — here they become lines you write into every template:

1. **Never paste sensitive data** (PHI, PII, confidential) into public tools without an approved policy.
2. **Human-in-the-loop** — AI drafts, a person reviews and owns the output (every card ends with a
   "check before sending" line).
3. **Verify facts and figures** — AI can be confidently wrong.
4. **Disclose where required** and respect confidentiality.
5. **Company standard over improvisation** — that is exactly what a template kit gives you.
